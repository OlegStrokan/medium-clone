# User service


## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
## Description

### This microservice is responsible for actions with users part.

### Events:

    - user_get_by_id
    - user_get
    - user_create
    - user_update
    - user_delete
